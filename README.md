# Palladium Global Resources Limited 🏗️

A premium, high-performance corporate landing page for **Palladium Global Resources Limited**, a Nigerian conglomerate focused on industrialization, infrastructure, and innovation.

## 🎨 Brand Identity
- **Primary Color:** Dark Olive Green (`#3f4d21`) - Represents stability and engineering growth.
- **Accent Color:** Safety Yellow (`#facc15`) - Represents industrial readiness and logistics.
- **Typography:** Inter (San-serif) for body; Playfair Display (Serif) for prestige sections.

## 🚀 Tech Stack
- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS 4.0
- **Deployment:** Vercel (CI/CD via GitHub)
- **Icons:** Heroicons / Lucide React

## 📂 Project Structure
- `/components`: Modular React components (Hero, About, Services, etc.)
- `/app`: Next.js App Router and Global Layout
- `/public`: High-resolution architectural and industrial assets

## 🛠️ Local Development
1. Clone the repo: `git clone https://github.com/Oper567/Palladium_Global_Resources_Limited.git`
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
